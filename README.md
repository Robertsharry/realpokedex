# RealPokedex

A free, feature-rich Pokedex with a battle team builder, type matchup explorer, and more. Built by a dad and Claude for his sons.

## Features

- **Pokedex Browser** - All 1025 Pokemon with search, type/generation filters, sorting, and favorites
- **Pokemon Detail Pages** - Animated stat bars, evolution chains, full move lists, type effectiveness, cry audio
- **Battle Team Builder** - Build teams of 6, real-time type coverage analysis, weakness detection, coverage scoring, smart suggestions
- **Type Matchup Explorer** - Interactive 18x18 type chart, dual-type calculator
- **Pokemon Compare** - Side-by-side stats with radar charts, type advantage analysis
- **Favorites & Collection** - Track your progress toward catching 'em all, generation breakdown

## Tech Stack

- Next.js 16 (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- Framer Motion
- Zustand (localStorage persistence)
- PokeAPI v2

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
